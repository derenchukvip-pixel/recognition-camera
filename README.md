# recognition_camera

MVP Flutter-приложение для осознанного потребления

## Основные возможности
- Захват изображения с камеры или галереи
- Сканирование штрихкода
- On-device AI (TFLite) для распознавания продукта
- Интеграция с Open Food Facts
- Облачный AI через бэкенд
- Кэширование информации о продуктах
- Базовый UI и поддержка подписки
- Дисклеймер при первом запуске с подтверждением согласия

## Структура проекта
- `lib/presentation` — UI-экраны, view-models, state
- `lib/domain` — модели и контракты
- `lib/data` — работа с API и источниками данных
- `lib/core` — конфигурация и инфраструктура

## Конфигурация
По умолчанию запросы отправляются на публичный backend. Можно переопределить:

- `RECOGNITION_BASE_URL` — базовый URL сервиса распознавания

## Запуск
1. Установите зависимости:
   ```sh
   flutter pub get
   ```
2. Запустите приложение на эмуляторе или устройстве:
   ```sh
   flutter run --dart-define=RECOGNITION_BASE_URL=https://your-server
   ```
